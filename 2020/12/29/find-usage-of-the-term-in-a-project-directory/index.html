Find usage of the term in a project directory

Recently I was using a terminal more often while I had VS Code open for editing my text files. VS Code search can be effective, but often times slows down he process. I am sure if I used a more powerful IDE that wouldn’t be an issue, but I recently found out I could use this command for searching my project repo or folders in a repo to find usage of a term!

grep -R {looking_for?} {directory} 

// example: find the usage of the NiceButton component in the src directory 
 
grep -R NiceButton src
